## 🌟 Overview

**All_In_One_Tool** is a powerful, all‑in‑one system utility that puts you in full control of your Windows PC.  
From deep cleaning junk files and disabling telemetry to real‑time monitoring of unwanted software,  
WinPulse Pro combines **12 essential tools** in a sleek, modern interface – built for power users and beginners alike.

---

## ✨ Features

### 📊 **Real‑Time Dashboard**
- Live CPU, memory, disk, and network graphs
- Top process list with **one‑click process kill**
- Instant memory boost

### 🧹 **Deep System Cleaner**
- Scan & clean temp files, browser cache, logs, prefetch, thumbnails, memory dumps & more
- Select exactly what to remove, see total reclaimable space
- Safely empty Recycle Bin

### 🔧 **Registry Cleaner & Fixer**
- Detect invalid uninstall entries, missing DLLs, orphaned startup items
- Automatic registry backup before fixes
- Colour‑coded severity levels

### ⚙️ **Advanced Service Manager**
- View all services with status, startup type, and risk analysis
- Filter by category (Running, Stopped, Recommendations)
- One‑click Stop / Start / Disable / Delete
- Powerful right‑click context menu

### 🛡️ **Firewall Control**
- Monitor active network connections with process details
- Block / unblock programs instantly
- Add custom rules, view listening ports
- Installed programs overview with block status

### 🚀 **Startup Manager**
- See **all** startup entries (Registry, Startup folder, Scheduled Tasks)
- Enable / disable items safely (backup & restore)
- Search and filter in real time

### 🗑️ **Software Uninstaller & Bloatware Killer**
- List every installed program (including hidden/system components)
- Uninstall with leftover file & folder cleanup
- Built‑in bloatware detection (Candy Crush, Xbox, McAfee…)

### 🔒 **Privacy Shield**
- Scan & enforce **8 critical privacy settings**:
  - Telemetry level, Cortana, Advertising ID, Wi‑Fi Sense, Find My Device, Activity History, Tailored Experiences, Inking & Typing
- Apply all privacy recommendations with one click

### ⚡ **AI Performance Booster**
- Suggests disabling heavy services, startup items, scheduled tasks
- Community‑curated recommendations for maximum speed
- Bulk apply with undo possibility

### 👁️ **Real‑Time Monitor**
- Background thread watches if previously uninstalled programs/services/tasks reappear
- Instant pop‑up alerts
- Keeps a history of all removals

### 📋 **Built‑in Logs**
- Every action is logged with timestamp
- Export, clear, or view logs directly in the app
